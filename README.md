# pyextract
Converts output from Multiscan and BioTek spectrometers to MPA readable files
## Instructions
1. Clone or Download
2. Run pyextract.py using Python 2.7
3. Select .txt or .exp file from Biotek or Multiscan spectrophotometer
4. Select output file
5. Click Convert
